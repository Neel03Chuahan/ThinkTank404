# ThinkTank404
Problem statement name -  AI-Powered Meeting Summarizer with Task Extraction & Smart Scheduling.

This prototype will includes audio/text meeting input, an AI-based transcription and NLP engine, task extraction with prioritization, and integration with task demonstrating complete automation from meeting input to scheduled tasks.

Unlike basic summarizers, it understands urgency and tone to prioritize tasks​

The system does not just summarize meetings — it actively manages outcomes,ensuring that important actions never slip through the cracks.​
